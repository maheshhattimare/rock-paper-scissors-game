# Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built with HTML, CSS, and JavaScript. This app allows users to choose an option, and the CPU randomly generates its answer to decide the result.

## Visit the Website

[Rock Paper Scissors Game](https://maheshhattimare.github.io/rock-paper-scissors-game/)

## Features

- User-friendly interface
- Interactive gameplay
- Random CPU choice generation
- Displays results and keeps score

## Technologies Used

- HTML
- CSS
- JavaScript

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/maheshhattimare/rock-paper-scissors-game.git
    ```
2. Navigate to the project directory:
    ```bash
    cd rock-paper-scissors-game
    ```
3. Open `index.html` in your web browser to start the application.

## Usage

1. Open the `index.html` file in your preferred web browser.
2. Click on a button to select your choice (Rock, Paper, or Scissors).
3. The CPU will randomly select its choice, and the result will be displayed.
